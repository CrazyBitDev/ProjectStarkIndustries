◆ prenotazioneModificabile()

bool prenotazioneModificabile ( Prenotazione *  prenotazione)

Function: prenotazioneModificabile

Controlla se una prenotazione è modificabile (o eliminabile) o meno. Una prenotazione è modificabile prima di due giorni l'inizio della mostra

Parametri
prenotazione: prenotazione da controllare se è modificabile
Restituisce
true se la prenotazione è modificabile, quindi il giorno corrente è antecedente a due giorni prima dell'inizio della mostra, altrimenti false
